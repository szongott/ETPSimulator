Required libraries
------------------

The following libraries are needed to run the simulator:

Bouncy Castle (Java cryptography APIs)
- tested with version jdk15on-148
- available here: https://www.bouncycastle.org/latest_releases.html


Apache Log4j 2
- available here: http://logging.apache.org/log4j/2.x/download.html


Mysql Connector
- available here: http://dev.mysql.com/downloads/connector/j/
